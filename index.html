このサイトについて

ライブ配信の聴き方

  1. 「ライブ配信」タブから空港を選ぶ
  2. 「LIVE」ボタンをタップして LiveATC.net を開く
  3. LiveATC のページでストリームを選んで再生

録音のアップロード方法 (Google Colab)

  1. リポジトリ内の colab/atc_recorder.ipynb を Google Colab で開く
  2. GitHub の Personal Access Token（repo スコープ）を設定
  3. 録音したい空港・周波数・時間を入力して実行
  4. 自動で LiveATC から録音し GitHub にアップロード
  5. Cloudflare Pages が自動デプロイ → このサイトに反映

Cloudflare Pages のセットアップ

  1. Cloudflare Pages でプロジェクトを作成
  2. GitHub リポジトリを連携
  3. ビルドコマンドは不要（静的サイト）
  4. 出力ディレクトリを / に設定
  5. push するたびに自動デプロイ

注意事項

  • LiveATC.net の利用規約に従って使用してください
  • 録音ファイルは aviation radio の受信記録です
  • 個人利用・学習目的のみ
